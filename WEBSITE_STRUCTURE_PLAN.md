## 🎨 DESIGN SYSTEM YANG DIGUNAKAN

### Colors
| Warna | Hex | Penggunaan |
|-------|-----|------------|
| Primary Blue | #1447E6 | CTA buttons, links |
| Brand Cyan | #00D9FF | Logo, accents, highlights |
| Background | #09090B | Main background |
| Surface | #18181B | Cards, containers |
| Border | #27272A | Borders, dividers |
| Text | #FFFFFF | All text |
| Text Secondary | #52525C | Disabled, helper text |
| Success | #009588 | Success states |
| Error | #E40014 | Error states |

### Typography
| Element | Size | Weight |
|---------|------|--------|
| H1 (Hero) | 60px | 800 |
| H2 (Section) | 36px | 600 |
| H3 (Card Title) | 30px | 700 |
| H4 (Subsection) | 24px | 700 |
| Body | 20px | 400 |
| Body Small | 16px | 400 |
| Label/Button | 15px | 500 |

### Components
| Component | Style |
|-----------|-------|
| Buttons | 36px height, 8px padding, 8px radius |
| Cards | #18181B bg, #27272A border, 10px radius |
| Inputs | #18181B bg, #27272A border, 8px radius |
| Shadows | Subtle, layered, cool-toned |

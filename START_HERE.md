## 🎯 Tugas Pertama

### Tambah Gambar
1. Siapkan gambar produk (12 buah)
2. Letakkan di folder: `assets/images/`
3. Nama file: `product-1.jpg`, `product-2.jpg`, dst.

### Update Informasi Perusahaan
1. Edit `pages/contact.html`
2. Ubah: Alamat, Telepon, Email
3. Edit `pages/about.html`
4. Ubah: Cerita perusahaan, Visi, Misi

### Ubah Warna
1. Edit `css/style.css`
2. Cari: `:root { --primary-color: #007BFF; }`
3. Ubah warna sesuai keinginan

## 🔧 Kustomisasi Cepat

### Ubah Judul Halaman
Edit file HTML, cari `<title>` dan ubah teks.

### Ubah Warna Tombol
Edit `css/style.css`, cari `.btn-primary` dan ubah `background-color`.

### Tambah Produk Baru
Edit `js/products.js`, tambahkan object baru ke array `productsData`.

### Ubah Teks Hero
Edit `index.html`, cari `<h1>` di section hero dan ubah teks.

## ❓ FAQ

### Q: Bagaimana menambah produk?
A: Edit `js/products.js`, tambahkan object ke array `productsData`.

### Q: Bagaimana mengubah warna?
A: Edit `css/style.css`, ubah CSS variables di `:root`.

### Q: Bagaimana menambah halaman?
A: Buat file HTML baru di folder `pages/`, link dari menu.

### Q: Bagaimana form kontak bekerja?
A: Form saat ini simulasi. Untuk real, setup backend API.

### Q: Bagaimana menambah gambar?
A: Letakkan di `assets/images/`, update path di HTML/JS.

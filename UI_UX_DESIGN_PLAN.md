# UI/UX Design Plan - Company Profile Website

## Project Overview
- **Tujuan**: Membuat website company profile dengan nama perushaaan Dronixsys yang profesional dan responsif
- **Tech Stack**: HTML, CSS, JavaScript
- **Theme**: Light Theme

## 7. ANIMASI YANG DIRENCANAKAN

| Tipe Animasi | Deskripsi | Durasi | Easing | Status |
|---|---|---|---|---|
| Fade-in | Elemen muncul dengan opacity | 0.5s | ease-in-out | ☐ |
| Slide-up | Elemen bergerak dari bawah ke atas | 0.6s | ease-out | ☐ |
| Slide-in (left/right) | Elemen bergerak dari samping | 0.6s | ease-out | ☐ |
| Stagger | Multiple elemen dengan delay | 0.1s per item | ease-out | ☐ |
| Hover lift | Elemen terangkat saat hover | 0.3s | ease-out | ☐ |
| Zoom | Elemen membesar saat hover | 0.3s | ease-out | ☐ |
| Pulse | Elemen berkedip/berdenyut | 1.5s | ease-in-out | ☐ |
| Shake | Elemen bergetar (error state) | 0.3s | ease-in-out | ☐ |
| Sticky | Navigation tetap di atas saat scroll | Instant | - | ☐ |
